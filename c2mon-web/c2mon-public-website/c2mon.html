Built for complex operational environments, where monitoring and control systems are asked to satisfy stringent reliability requirements, C2MON is a monitoring system that can adapt to changes in its environment and meet requests for new functionalities. C2MON manages to combine maintainability and high-availability within a portable architecture.

C2MON is used at CERN. The world's largest particle physics laboratory.

The technical infrastructure operators (BE-OP-TI) have chosen C2MON to monitor CERN's technical services 24 hours a day, 365 days a year.

C2MON is RELIABLE*


When youâ€™re trying to understand the building blocks of matter, the origin of the universe and everything in between, system downtime is not an option.

- C2MON is designed to provide 24/7 availability
- JMS transactions ensure that no message is lost and all the updates are written to the cache
- Multi server configuration allows for rolling updates and provides a failover mechanism

C2MON is SCALABLE, at all levels.


- Scalable JMS Middleware
- Unlimited Data Acquisition processes
- Multi server configuration means if the limit of the current system is reached servers can be scaled horizontally

C2MON is FLEXIBLE, and designed from the start to fit multiple monitoring scenarios.

- Customizable Server
- Data Acquisition using a variety of protocols
- Operational Database Independence
- Easily reused Client API (actual GUI implementation also available)
